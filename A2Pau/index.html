Guía de Git para Principiantes

Bienvenido a la guía para aprender Git desde cero.

Recursos Básicos

  • Pro Git Book: Este libro es una excelente introducción a Git. Aprenderás los conceptos básicos y avanzados de Git.
  • Try Git: Una plataforma interactiva que te permite aprender Git directamente desde tu navegador.

Tutoriales en Video

  • Git Tutorial for Beginners: Command-Line Fundamentals: Un video introductorio que cubre los comandos básicos de Git.
  • Git & GitHub Crash Course For Beginners: Un video que te guiará a través de Git y GitHub para principiantes.

Referencia de Comandos

  • Git Documentation: La documentación oficial de Git, donde encontrarás una referencia completa de todos los comandos de Git.

Colaboración y Ramificación

  • Comparing Workflows: Comprende las diferentes estrategias de flujo de trabajo en Git.
  • A Successful Git Branching Model: Un modelo de ramificación popular llamado Gitflow.
